# How to use drive images in html

Sometimes we need to use many photos in our projects, but we have a problem: the storage.

If you are developing a personal project and you don't want to pay hostings with more extra storage, you can use this method.

Obviusly this can affect the performance of the project.

With that said, let's get started.


## Upload images in our Google Drive
Remember if we have a basic account, we can store 15Gb free.

First upload the images (preferably in a folder)

After that, we have to give public permissions to the images:

![Example](/images/assets/how-to-use-drive-images-in-html/permissions.png)

## Get link automatically
Now we will get the link of the image.

This tools return a converted link that we can use in our HTML.
  - First copy the link of the image and paste it here.
  - Don't paste it in your HTML yet, this won't work.
  - Paste your link here and press the button.
  - Copy the generated link and put in your HTML.

<ImgDriveConverter/>
<br/>



## Get link manually

First copy the link of the image.

After, modify it to turn this: <br/>'https://drive.google.com/file/d/<b>12HLf2crewvXmzZ66XxNK5tvDFz1Ps3AC</b>/view?usp=sharing' <br/><br/>to this: <br/> 'https://drive.google.com/uc?export=view&id=<b>12HLf2crewvXmzZ66XxNK5tvDFz1Ps3AC</b>'.
